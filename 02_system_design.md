# 02. 시스템 설계 문서

## 1. 문제 정의

사회복지사 2급 및 정신건강사회복지사 준비 학습자는 강의 교안을 바탕으로 핵심 개념, 시험 포인트, 현장 적용점을 정리해야 한다. 그러나 강의 교안은 개념, 연도, 통계, 제도 변화가 함께 제시되어 초보 학습자가 핵심 구조를 빠르게 파악하기 어렵다.

따라서 본 프로젝트는 **사회복지 강의 교안을 입력하면 복습 및 시험 대비용 학습 노트를 자동으로 생성하는 LLM 프롬프트 시스템**을 설계하는 것을 목표로 한다.

---

## 2. 타깃 사용자

| 구분 | 내용 |
|---|---|
| 주요 사용자 | 사회복지사 2급 과정을 수강하는 학습자 |
| 확장 사용자 | 정신건강사회복지사 진로를 준비하는 학습자 |
| 사용 목적 | 강의 교안 복습, 시험 대비, 개념 정리, 현장 적용 이해 |
| 사용 상황 | 강의 수강 후 교안 내용을 빠르게 학습 노트로 정리하고 싶을 때 |

---

## 3. 업무 과업 정의

### 업무 과업

**사회복지 강의 교안 기반 학습 노트 자동 생성**

### 입력 데이터 형태

- 과목명
- 주차/차시
- 강의주제
- 강의 교안 본문
- 정리 목적

### 출력 규격

- 한 문장 핵심 요약
- 강의 흐름
- 주요 개념 정리
- 시험 포인트
- 현장 적용
- 헷갈리기 쉬운 부분
- 확인 필요 사항
- 최종 정리

---

## 4. 업무 과업 입력 템플릿

```text
[업무 과업]
사회복지 강의 교안 기반 학습 노트 정리

[과목명]
예: 장애인복지론

[주차/차시]
예: 1주차 1교시

[강의주제]
예: 장애의 개념과 범주

[강의 교안]
여기에 강의 교안 내용을 붙여넣기

[정리 목적]
예: 복습 / 시험 대비 / 개념 이해 / 현장 적용

[원하는 결과]
핵심 요약 / 강의 흐름 / 주요 개념 / 시험 포인트 / 현장 적용 / 헷갈리는 부분 / 확인 필요 사항 / 최종 정리

[톤]
대학생 학습용, 객관적, 명확함

[금지]
교안에 없는 학자명, 법령명, 정책명, 수치, 사례를 임의로 추가하지 않기

[확인 질문 규칙]
정보가 부족하면 최대 3개까지 확인 질문 후 정리하기
